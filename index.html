Semantic HTML

This website is intended to help inform interested parties about the importance of semantic HTML.

Why is it important?

Semantic HTML provides many benefits to people, devices, and software reading our HTML documents, and how they're able to comprehend them.

How to apply Semantic HTML to a document?

  1. Start by pasting just the text of the content into the body of a new HTML document.

  2. Determine the major components of your content?

    • Main Header - What is your heading content?
    • Main - What is the main content?
  3. Once you determined the major components, determine if there are any sub-components.

Structured HTML and Semantic HTML Fall Hand-in-Hand

Structured HTML is the practice of formatting the HTML in the correct order of intended digestion. This means, that the content is organized in the order it was expected to be read.
